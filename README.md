# ReconVault

ReconVault is a penetration testing recon automation CLI that creates a clean workspace per target and runs repeatable scans (starting with Nmap).

## Features (MVP)
- Target workspace creation
- Nmap basic scan output saved per target
- Clean folder structure for scans/evidence/notes

## Installation
```bash
git clone https://github.com/Arpitraj93/reconvault
cd reconvault
pip3 install -r requirements.txt
